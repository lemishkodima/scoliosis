# План створення лендінгу

## Обсяг MVP

Перший реліз має бути односторінковим сайтом для `scoliosis.ua` з формою заявки на членство. Архітектура повинна дозволяти майбутнє розширення до багатосторінкового сайту або вебзастосунку.

MVP включає:

- Перший екран із 3D-hero на основі логотипу.
- Блок переваг членства.
- Блок місії.
- Блок "Для кого".
- Блок "Членство відкриває".
- Блок "Як стати членом".
- Фінальний CTA.
- Форму заявки.
- Інтеграції заявки: Google Sheets, email, Telegram bot.
- Базову SEO-структуру.
- Адаптивність для mobile/tablet/desktop.

## Технічна стратегія

Оскільки проєкт планується розширювати, краще не робити одноразовий HTML без структури. Рекомендований стек:

- Next.js або Astro для швидкого лендінгу з можливістю масштабування.
- TypeScript для форми та інтеграцій.
- CSS modules або Tailwind CSS, залежно від майбутніх потреб.
- Three.js або React Three Fiber для 3D-hero.
- Serverless API route для заявки.

Якщо пріоритет - максимально швидкий запуск, можна стартувати з Astro. Якщо пріоритет - майбутній кабінет, адмінка і складніші інтеграції, краще Next.js.

Рекомендація: Next.js, бо форма, інтеграції та майбутній membership-функціонал природно лягають у цей стек.

## Візуальна система

### Кольори

Отримана палітра:

- `#FFFFFF` - основний фон.
- `#DFE4F5` - світлий медичний фон, секції, підкладки.
- `#A2B4E2` - мʼякий акцент, підсвітки, secondary UI.
- `#515BAB` - основний акцент, CTA, активні елементи.
- `#17325C` - темний текст, глибокі акценти, навігація.

Примітка: у побажанні колір був записаний як `#17325С`, де остання літера схожа на кириличну `С`. У коді потрібно використовувати латинську `C`: `#17325C`.

### Шрифт

Зазначено "Body Text Large". Це схоже на назву текстового стилю з Figma, а не на назву шрифту.

Рекомендація для MVP:

- Основний шрифт: Inter або Manrope.
- Якщо у вихідному брендингу є конкретний font-family, його потрібно підставити після уточнення.
- Текстові стилі: великий читабельний body, сильні заголовки, без надмірно декоративних шрифтів.

### Форма

UI має бути спокійним і довірливим:

- Поля з чіткими label.
- Мінімум зайвого тексту.
- Стан відправлення, успіху та помилки.
- Захист від випадкового повторного submit.
- Honeypot або Turnstile/reCAPTCHA на наступному етапі.

## 3D-hero з логотипу

### Ціль

Перший екран має створювати "вау ефект" без втрати медичної довіри. 3D має підсилювати бренд, а не виглядати як випадкова декоративна анімація.

### Концепція

У центрі першого екрана - обʼємна інтерпретація логотипу:

- Силует спини як напівпрозора обʼємна форма.
- Хребет як світла 3D-структура з мʼяким підсвічуванням.
- Геометричні лінії плеча як тонка медична сітка.
- Повільне parallax-обертання при русі миші.
- Легка scroll-анімація: герой відʼїжджає, секції проявляються.

### Реалізація для MVP

Перший етап:

- Використати наданий PNG логотип як основу.
- Створити layered hero: PNG + CSS depth + WebGL particles/lines.
- Додати інтерактивний parallax.
- Забезпечити fallback для мобільних і слабких пристроїв.

Другий етап:

- Векторизувати логотип у SVG.
- Розкласти логотип на шари: силует, хребет, геометрія, текст.
- Екструдувати ключові шари у Three.js.
- Оптимізувати модель як glTF/GLB.

Третій етап:

- Створити повноцінну 3D-модель із брендовими матеріалами.
- Додати контрольовану анімацію вступу.
- Перевірити продуктивність на mobile.

## Структура сторінки

### Header

Елементи:

- Логотип.
- Навігація: Переваги, Місія, Для кого, Як вступити.
- CTA: Стати членом.

Header має бути легким, sticky після першого scroll.

### Hero

Текст:

"Разом формуємо майбутнє консервативного лікування сколіозу в Україні"

Підтекст:

"Ми обʼєднуємо спеціалістів, які прагнуть розвиватися, навчатися один в одного та впроваджувати сучасні світові стандарти лікування."

CTA:

"Стати членом"

Візуал:

3D-логотип або layered 3D-hero.

### Не просто членство

Задача блоку: пояснити, що асоціація - це не формальність, а професійне середовище.

Текст потрібно виправити:

- Було: "Бути учасникрм"
- Має бути: "Бути учасником"

### Що ви отримуєте

4 benefits:

- Навчання без кордонів.
- Спільноту однодумців.
- Міжнародний розвиток.
- Нові можливості.

Іконки краще робити через lucide icons або власну лінійну систему, а не emoji. Emoji можна залишити тільки в чорновому контенті, але в фінальному UI краще професійні іконки.

### Наша місія

Задача блоку: підсилити довіру і пояснити суспільну користь.

Можна використати фото або split-layout з якісним зображенням реабілітаційної роботи.

### Для кого

Формат:

- Список із check icons.
- Стисла подача.
- Можливо, поруч фото професійної роботи або мʼяка брендова графіка.

### Членство відкриває

Формат:

- Великий типографічний блок.
- 4 короткі рядки.
- Візуальний ритм через progressive reveal на scroll.

### Як стати членом

3 кроки:

1. Заповніть заявку.
2. Підтвердіть членство.
3. Отримайте доступ до можливостей асоціації.

Формат:

- Горизонтальна timeline на desktop.
- Вертикальна timeline на mobile.

### Заявка

Поля форми MVP:

- Імʼя та прізвище.
- Email.
- Телефон.
- Місто.
- Спеціалізація.
- Місце роботи або організація.
- Досвід роботи зі сколіозом.
- Категорія: лікар, фізичний терапевт, реабілітолог, ортезист, викладач, студент, інше.
- Повідомлення.
- Згода на обробку персональних даних.
